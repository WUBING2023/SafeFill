# 命令入口清单

> 本文档列出SafeFill所有可执行命令。每条命令都说明了用途、前提、输入输出和安全属性。

---

## 命令总览

| 序号 | 阶段 | 命令 | 联网 | 改原始 | 改 vault |
|------|------|------|------|--------|----------|
| 1 | 第 1 阶段 | `extract_candidates.py` | 否 | 否 | 否 |
| 2 | 第 1 阶段 | `save_confirmed_profile.py` | 否 | 否 | ✅ 写入 |
| 3 | 填写检查 | `form_review.py` | 否 | 否 | 否 |

| 5 | API 检查 | `api_assist.py check` | 否 | 否 | 否 |

## 1. 提取候选资料

```bash
python D:\SafeFill\app\extract_candidates.py
```

| 属性 | 值 |
|------|-----|
| 用途 | 从 input_forms 中的旧 .docx/.xlsx 提取候选个人信息 |
| 前提 | `input_forms\` 中有 .docx 或 .xlsx 文件 |
| 输入目录 | `input_forms\` |
| 输出目录 | `candidate_reviews\` |
| 联网 | 否 |
| API | 否 |
| 修改原始文件 | 否 |
| 修改 vault | 否 |

---

## 2. 保存确认资料

```bash
python D:\SafeFill\app\save_confirmed_profile.py
```

| 属性 | 值 |
|------|-----|
| 用途 | 读取最新候选资料，询问用户确认后保存到唯一个人信息库 `vault\profile.json` |
| 命令 | 默认保存流程 / `show` 查看个人信息表 |
| 前提 | `candidate_reviews\latest_candidate.json` 存在 |
| 输入目录 | `candidate_reviews\` |
| 输出目录 | `vault\` + `vault\backups\`（覆盖前自动备份） |
| 联网 | 否 |
| API | 否 |
| 修改原始文件 | 否 |
| 修改 vault | 是，仅用户确认后修改 `vault\profile.json` |

---

## 3. 填写检查（FormReview）

```bash
python D:\SafeFill\app\form_review.py
python D:\SafeFill\app\form_review.py --no-open
```

| 属性 | 值 |
|------|-----|
| 用途 | 自动填写新表格 + 启动网页检查（合并入口）。浏览器会自动打开；如未打开，可手动访问 http://127.0.0.1:8787/ |
| 前提 | `vault\profile.json` 存在；`new_forms\` 中有 .docx/.xlsx |
| --no-open | 禁用自动打开浏览器，仅打印网址 |
| 输入目录 | `new_forms\` + `vault\` |
| 输出目录 | `review_html\` + `draft_outputs\` + `filling_reports\` + `review_results\` |
| 访问地址 | 浏览器自动打开 `http://127.0.0.1:8787/`（`--no-open` 禁用） |
| 联网 | 否 |
| API | 否 |
| 修改原始文件 | 否 |
| 修改 vault | 否 |

> 内部调用 `fill_form_draft.py` 和 `review_server.py`，不建议用户单独运行这两个脚本。
> 停止服务：按 `Ctrl+C`。

## 7. 隐私清理预览

```bash
python D:\SafeFill\app\cleaner.py preview
```

| 属性 | 值 |
|------|-----|
| 用途 | 预览可清理的旧表、草稿、报告、网页检查结果等过程产物 |
| 前提 | 一轮测试或正式填表完成 |
| 默认保留 | `vault\`、`final_outputs\`、`app\`、`skills\`、`docs\` |
| 联网 | 否 |
| API | 否 |
| 修改原始文件 | 否，preview 不删除 |
| 修改 vault | 否 |

真正清理必须由用户确认：

```bash
python D:\SafeFill\app\cleaner.py clean --confirm CLEAN
```

Cleaner 不会默认执行删除。

## SafeFill-FinalExport html_review 说明

`python D:\SafeFill\app\export_final.py` 支持两类 `review_result`：

1. ReviewUI v2 生成的 `mode=html_review`：根据网页中可编辑表格的 `tables` 内容逐格写回最终 Word / Excel。
2. 旧版字段确认结果：继续按 `confirmed_fields`、`modified_fields` 等字段导出。

无论哪种模式，FinalExport 都只复制并修改 `final_outputs\` 中的最终文件副本，不修改原始文件、草稿文件或 vault。
