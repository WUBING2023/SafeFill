# 第 1.2 阶段 — 专业与研究方向字段拆分

## 为什么要拆分

研究生培养计划文档中同时出现了"专业"和"研究方向"：

- 专业：兽医
- 研究方向：动物生物药物研发与应用

这两个是不同的信息。项目申报表可能分别要求填写"专业"和"研究方向"。

拆分前，"专业"和"研究方向"都映射到 `research_area`，去重时先出现的"兽医"覆盖了后出现的"动物生物药物研发与应用"，导致研究方向信息丢失。

## 拆分后字段映射

| 中文字段名 | 映射字段 key | 说明 |
|-----------|-------------|------|
| 专业、所学专业、攻读专业、学科专业 | `major` | 学科或培养专业 |
| 研究方向、研究领域、研究内容 | `research_area` | 具体研究课题方向 |
| 专业方向、学科方向、从事专业 | `research_area` | 研究方向类（非专业名） |

## 本次修改内容

| 文件 | 修改 |
|------|------|
| `profile_template.json` | 新增 `major` 字段 |
