# 数据字段清单

|----------|----------|---------------|-----------------|----------------------|------|
| 姓名 | 否 | ✅ 是 | ✅ 是 | ✅ 是（脱敏后） | 基础身份字段 |
| 性别 | 否 | ✅ 是 | ✅ 是 | ✅ 是 | 基础身份字段 |

| 身份证号 | **🔴 敏感** | ✅ 是 | ✅ 是 | **❌ 禁止发送** | 最高保护级别；日志中不记录内容 |
| 手机号 | **🔴 敏感** | ✅ 是 | ✅ 是 | **❌ 禁止发送** | 最高保护级别；日志中不记录内容 |

## 符号说明

| 符号 | 含义 |
|------|------|
| ✅ 是 | 支持或允许 |
| ❌ 禁止 | 绝对不允许 |
| ⚠️ 需确认 | 需要用户逐项确认后才能执行 |
| 🔴 敏感 | 敏感信息，最高保护级别 |
| 🟡 需确认 | 涉及隐私或未公开信息，需用户确认 |

## 敏感字段处理细则

### 身份证号
- **存储**：加密存储于 `vault/profiles/`
- **自动填写**：允许自动填入表格对应字段
- **API**：**禁止发送**，调用 API 前自动拦截
- **日志**：仅记录"身份证号字段已填写"，不记录具体号码
- **展示**：本地网页展示时默认部分脱敏（如 `320***********1234`）

### 手机号
- **存储**：加密存储于 `vault/profiles/`
- **自动填写**：允许自动填入表格对应字段
- **API**：**禁止发送**，调用 API 前自动拦截
- **日志**：仅记录"手机号字段已填写"，不记录具体号码
- **展示**：本地网页展示时默认部分脱敏（如 `138****1234`）

### 通讯地址
- **存储**：加密存储于 `vault/profiles/`
- **自动填写**：允许自动填入表格对应字段
- **API**：**禁止发送**，调用 API 前自动拦截
- **日志**：不记录通讯地址内容
- **展示**：本地网页展示时默认完整展示（地址通常不脱敏展示，但禁止发送给 API）

### 证件照路径
- **存储**：仅存储文件路径于 `vault/profiles/`，实际图片文件存放于 `vault/photos/`
- **自动填写**：第一版不自动插入图片到表格，仅记录路径
- **API**：**禁止发送**（路径和图片内容均禁止）
- **日志**：仅记录证件照是否已配置
- **第一版限制**：不处理图片内容，后续版本再支持将证件照插入表格

### 常用项目经历
- **存储**：加密存储于 `vault/profiles/`
- **自动填写**：需用户逐项确认后，才自动填入匹配的表格字段。不同项目可能涉及不同的经历条目，用户应能选择哪些经历用于当前表格。
- **API**：**禁止发送**（涉及未公开项目信息）
- **展示**：本地网页中完整展示，供用户核对

### 常用个人简介
- **存储**：加密存储于 `vault/profiles/`
- **自动填写**：需用户确认后，才自动填入匹配的表格字段。
- **API**：**禁止发送**（完整个人履历属于敏感信息）
- **展示**：本地网页中完整展示，供用户核对
