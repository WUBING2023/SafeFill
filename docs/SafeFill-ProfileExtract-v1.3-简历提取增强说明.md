# SafeFill-ProfileExtract v1.3 — 简历提取增强

## 增强内容

### 1. 字段名归一化

新增 `normalize_field_name()` 函数，自动去掉字段名中的空格、Tab、全角/半角冒号。

| 原始文本 | 归一化后 |
|----------|----------|
| `姓 名` | `姓名` |
| `电 话：` | `电话` |
| `邮 箱:` | `邮箱` |
| `年 龄` | `年龄` |

### 2. 模糊匹配

当别名长度 >= 2 时，自动生成允许字符间插入空格的模糊正则：

```
"电话" → 电\s*话\s*[：:\t]\s*
```

可匹配：
- `电话：137xxxx`（正常）
- `电 话：137xxxx`（含空格）
- `电\t话：137xxxx`（含 Tab）

### 3. 简历专用别名

| 新增别名 | 映射字段 | 说明 |
|----------|----------|------|
| 现居地、现住址、居住地、所在地 | `address` | 简历常见地址字段 |
| 就读专业 | `major` | 教育经历中的专业 |
| 教育背景、教育经历 | `education` | 学历字段 |
| 年龄 | `age` | 新增字段，仅提取展示不入库 |

### 4. 个人信息保护

- 手机号：Markdown 和日志中脱敏（`137****1185`）
- 邮箱：当前视为非敏感，正常显示
- 地址：Markdown 中脱敏（前 6 字 + `***`）
