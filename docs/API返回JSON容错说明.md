# SafeFill API JSON 容错解析

## 概述

`api_assist.py` 中的 `robust_json_loads()` 提供渐进式 JSON 解析容错，处理大模型返回的轻微语法错误。

## 解析顺序

| 步骤 | 方法 | 说明 |
|------|------|------|
| 1 | `json.loads(text)` | 直接解析 |
| 2 | 提取 \`\`\`json 代码块 | 处理 Markdown 围栏 |
| 3 | 正则提取 JSON 对象/数组 | 去掉前后解释文字 |
| 4 | 轻量修复后解析 | 去尾逗号、替换中文引号 |
| 5 | 全文修复后解析 | 同上，范围更广 |

## 修复规则（仅安全操作）

- 去掉尾逗号：`,}` → `}`, `,]` → `]`
- 替换中文标点：`"` `"` → `"`, `'` `'` → `'`, `，` → `,`
- 数组自动包装：`form_fill_plan` 返回的 `[{...}]` 自动包装为 `{"writes": [...]}`
- 提取嵌在自然语言中的 JSON

## 不做的事

- 不猜字段值
- 不补缺失结构
- 不执行 eval
- 不引入外部依赖

## 追踪

修复成功时 API trace 标记：
- `json_repaired: true`
- `json_repair_message: "修复方式说明"`
