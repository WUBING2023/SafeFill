# SafeFill-FormReview 表格结构增强说明

## 目标

提高新表格自动填写成功率，尤其是常见 Word/Excel 空白表格结构。

## 新增支持

### 1. 字段在上一行，空格在下一行

示例：

```
姓名 | 手机号 | 导师
____ | ____   | ____
```

现在会把 vault 中的值写入下一行对应单元格。

### 2. 同一单元格冒号占位

示例：

```
姓名：____
导师：
```

现在会写成：

```
姓名：张三
导师：李四
```

### 3. 自定义字段

如果 vault 的 `custom_fields` 中有字段，也能填写：

- 民族
- 政治面貌
- 导师
- 学号
- 爱好

## 安全边界

1. 不修改 `new_forms\` 原始文件。
2. 只修改 `draft_outputs\` 中的草稿副本。
3. 已有内容不覆盖。
4. 合并单元格非左上角不写入。
5. 仍需要用户在网页中检查并点击“保存检查结果”。
6. FinalExport 只读取用户保存后的 `review_result`。

## 验证结果

已用临时 Word / Excel 文件验证：

- Word 同单元格冒号占位：通过。
- Word 字段在上一行：通过。
- Excel 同单元格冒号占位：通过。
- Excel 字段在上一行：通过。
